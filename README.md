# Flux CLI # 
**Flux** is a **prototype** custom CLI (command line interface) for managing and editing a personal configuration system. It provides intuitive subcommands for viewing, editing, and resetting configuration values with helpful matching for user friendly input

## Features 
- View, set, get, & reset configuration settings using `flux`   
- Command history tracking using `history`  
- Help system using `help`  
- Automatic fuzzy command matching system  
- Echo and Clear commands for CLI convenience  
- Simple JSON based storage for configurations  

## Commands 
`help` | Show a list of available commands  
`exit` | Exit the CLI     
`echo` | Print a message to the terminal  
`clear` | Clear the terminal screen  
`flux` | Manage configuration settings  
`history` | Show previously used commands  

## Installation
1. Clone the repository:
    `git clone clone https://github.com/danieljf10/Flux-CLI-Prototype`  
    `cd Flux-CLI` 

## Usage 
1. To run the program, please type:
    `python .\core\cli.py`
